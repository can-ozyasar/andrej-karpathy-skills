# Andrej Karpathy Skills

LLM tabanlı kodlama akışlarını iyileştirmek için CLAUDE.md odaklı yönerge koleksiyonu.

## Bu Repo Ne İçin Var?
LLM destekli kodlama araçlarının daha tutarlı çalışması için yönerge ve davranış denemelerini toplamak amacıyla oluşturuldu.

Bu README'nin amacı; repoya ilk kez gelen birinin projenin neden açıldığını, içinde ne bulunduğunu ve nereden başlaması gerektiğini hızlıca anlamasını sağlamaktır.

## İçerik ve Kapsam
Bu repoda öne çıkan içerikler şunlardır:
- Kodlama asistanı davranışlarını yönlendiren tekil doküman yaklaşımı
- Çok dilli README içeriği
- Prompt ve çalışma prensibi denemeleri için referans repo
- Repo amacını ve sonraki geliştirme adımlarını açıklayan başlangıç dokümantasyonu
